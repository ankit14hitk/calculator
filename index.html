Basic Instructions before use


1.The addition/plus is determined by + sign, Example: 24+32=56.


2.The subtraction is determined by - sign, Example: 49-24=25.


3.The Multiplication is determined by * sign. Example: 12*2=24.


4.The division/divide is determined by / sign, Example: 48/6=8.


5.The % sign stands for percentage sign using current and next entry.


6.The AC button stands for All clear.


7.The Del button stands for Delete.
